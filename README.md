## Stack

Plain HTML, CSS, and vanilla JavaScript — no build step required.

- `index.html` — markup, all sections
- `styles.css` — design system (tea-green palette, responsive)
- `script.js` — navbar, mobile menu, reveal animations, animated counters & metric bars

## Run locally

Just open `index.html` in a browser, or serve the folder:

```bash
python -m http.server 8080
# then visit http://localhost:8080
```

## Deploy on GitHub Pages

1. Push to GitHub (already done if you are reading this on the repo).
2. Repo **Settings → Pages → Source = `main` branch / root**.
3. Visit `https://<username>.github.io/teavision-website/`.
